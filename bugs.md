# LiciTrack — Bugs

> Problemas conocidos a corregir. Agregá cada bug con suficiente contexto para reproducirlo.

---

## Formato

```
### [Título corto del bug]
- **Dónde:** página o componente afectado
- **Cómo reproducir:** pasos para verlo
- **Comportamiento actual:** qué pasa
- **Comportamiento esperado:** qué debería pasar
- **Prioridad:** alta / media / baja
```
